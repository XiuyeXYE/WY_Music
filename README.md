# WY_Music
万能播放器

##  支持 x86 cpu arm架构
##  Android 基于：FFmpeg+OpenSL+OpenGL+Mediacodec 的视频播放SDK，可播放网络、本地和广播等流媒。
##  支持播放VR360度全景视频
### 1、支持当前播放页直接切换播放源
### 2、优先选用GPU解码，解码速度更快
### 3、支持软解码和硬解码
### 4、直接用mediacodec解码avpacket数据。
### 5、支持用于传统直播和VR直播
### 6、底层使用FFmpeg解码支持各种格式的音视频播放
### 7、支持rtmp,http,hls等多种流媒体协议
